## 🧩 O que está ligado
- **Código:** GitHub → https://github.com/Gui-S-1/OSdudu
- **Banco de dados:** Supabase, projeto **osdudu** (tabelas, permissões, realtime e os 4 logins já criados)
- **Publicação:** Vercel, projeto **o-sdudu** (variáveis de ambiente já configuradas)

Qualquer alteração que for enviada para o GitHub (branch `main`) a Vercel publica sozinha.

## 📲 Quer um APK de verdade?
Entre em **https://www.pwabuilder.com**, cole `https://o-sdudu.vercel.app`
e clique em **Package for Android** — ele gera o arquivo `.apk` para instalar.
